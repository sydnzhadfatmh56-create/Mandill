# To-Do List Application

یک برنامه‌ی **To-Do List** حرفه‌ای با قابلیت ذخیره‌سازی محلی (Local Storage).

## ✨ ویژگی‌ها

- ✅ افزودن، ویرایش، حذف کارها
- 💾 ذخیره‌سازی خودکار در Local Storage
- 🎯 علامت‌گذاری کارهای انجام شده
- 🔍 جستجو و فیلتر کردن
- 📊 نمایش تعداد کارهای انجام شده
- 🎨 رابط کاربری زیبا و حرفه‌ای
- 📱 طراحی واکنش‌پذیر (Responsive)
- 🌙 حالت Dark Mode

## 🛠️ تکنولوژی‌ها

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage API

## 📋 نحوه استفاده

1. باز کردن `index.html` در مرورگر
2. نوشتن کار جدید در فیلد ورودی
3. کلیک بر دکمه "افزودن"
4. کارها به صورت خودکار ذخیره می‌شوند
5. برای تکمیل کار روی آن کلیک کنید
6. برای حذف، دکمه حذف را بزنید
